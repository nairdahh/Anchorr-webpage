Why Choose Anchorr?

Discord Integration

Manage media requests and server notifications directly from Discord with intuitive slash commands.

Quick Start Guide

1

Clone the Repository

Get the source code from GitHub and run with Node.js.

git clone https://github.com/nairdahh/Anchorr.git cd Anchorr npm install node app.js
2

Configure via Web UI

Open the web interface and configure your Discord bot credentials, API keys, and Jellyseerr details.

http://localhost:8282
3

Set Up Webhooks

Add the webhook URL to your Jellyfin server for real-time notifications.

http://<your-host-ip>:8282/jellyfin-webhook
4

Invite Bot to Server

Generate an invite link in the Discord Developer Portal and add the bot to your server. Start enjoying Anchorr!

Alternative: Docker Installation

Prefer running in a container? Use Docker or Docker Compose for a quick setup.

docker compose up -d --build

For a fully detailed guide and advanced configuration, please visit the GitHub page.
