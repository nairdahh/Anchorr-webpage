Public Anchorr Bot

This is the public version of the Anchorr Discord bot. You can invite it directly to your server using the link below.

Invite Bot to Server Dashboard

Standalone & Easy to Use

Zero-Setup Solution

A direct, standalone implementation. No self-hosting required, allowing you to implement the service with minimal configuration.

Intuitive Web Dashboard

Once invited, the bot is configured entirely through a clean web interface, centralizing all your settings.

Efficient Management

With all settings in one place, server management becomes simpler and more efficient than ever.

1

Invite and Initial Setup

Add the bot to your server and type the command /setup. You will receive a private link to the web dashboard.

2

Connect to Jellyfin

Access the provided link to connect your bot to your Jellyfin and Jellyseerr instance, and customize settings, such as notification channels and embed colors.

3

Use Commands

Once configured, server members can use the /request and /search commands directly in Discord to find and request media content.

4

Enjoy Automatic Notifications

The bot will automatically send notifications to the designated channel when new content is added to Jellyfin, keeping the community updated.

Prefer a self-hosted version?

If you want complete control over your data and advanced customization options, you can host Anchorr on your own infrastructure. Visit the main page for the installation guide.
